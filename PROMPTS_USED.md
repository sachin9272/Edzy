## Engineering Prompts Used

1. "Scaffold a Next.js 14 App Router workspace with Turbopack, configuring strict TypeScript architectures and a centralized React Context layer for global modal state orchestration."
2. "Architect a MongoDB data layer defining a `Student` schema (name, unique referralCode, totalSpent) relationally bound to an `Order` model (studentId, snackId, quantity, payableAmount) to enforce strict data integrity."
3. "Design a robust Express & Mongoose backend API that exposes scalable RESTful endpoints, implementing relational `.populate()` schemas, rigorous payload validation, and granular HTTP exception bubbling."
4. "Abstract the application's aesthetic layer by engineering a rigorous design system, mapping dynamic CSS variables to strict design tokens to enforce pixel-perfect, scalable dark-theme consistency across all UI components."
5. "Integrate `@tanstack/react-query` to abstract our asynchronous network state, handle strategic background invalidations natively, and enforce reliable UI states across the dashboard."
6. "Engineer a dynamic, glass-morphic Sidebar navigation component utilizing a mobile-first animated drawer pattern, backdrop blurs, and strict body scroll-locks on open."
7. "Develop a resilient client-side persistence strategy for recent order histories utilizing lightweight `localStorage` hooks to optimize cold-start UX and reduce redundant network payloads."
8. "Implement a robust, asynchronous error boundary matrix to intelligently trap and recover from edge-case network failures, utilizing declarative fallback UI patterns without compromising the global component tree."
9. "Optimize the React hydration and rendering pipeline by aggressively memoizing complex component subtrees and data transforms, eliminating redundant reconciliation cycles down the DOM."
